Today was all about getting the foundation set up.

## ✅ What got done today

- Set up Roblox Studio and created the project
- Built out a clean folder structure (ServerScriptService, ReplicatedStorage, etc.)
- Created core scripts: `GameManager`, `EnemySpawner`, `RoundManager`, `ClientMain`

Nothing fancy yet, but this is the part that makes everything else easier later. The goal was just to get something organized and playable, and that's done.

## 🧠 Biggest Takeaway

Keep it simple. Don't overbuild. Just get something working.
